# サンプルブログ記事

これは**MDX**形式で書かれたサンプルブログ記事です。

## 特徴

- MarkdownとJSXの両方を使用できます
- Reactコンポーネントを埋め込むことができます
- コードハイライトが利用できます

## コード例

```typescript
function greet(name: string) {
  return `Hello, ${name}!`;
}
```

## まとめ

このように、MDXを使用することで、ブログ記事内でインタラクティブなコンテンツを作成できます。
